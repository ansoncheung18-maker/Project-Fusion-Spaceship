## 2. 關閉程序

### 2.1 關閉順序

| 步驟 | 動作 | 說明 |
|:---|:---|:---|
| 1 | 停止推進 | 關閉反應爐、磁噴嘴 |
| 2 | 停止發電 | 斷開所有系統 |
| 3 | 冷卻反應爐 | 繼續散熱至安全溫度 |
| 4 | 排空燃料 | 回收剩餘氘 |
| 5 | 排空冷卻劑 | 回收水（屏蔽層） |
| 6 | 關閉控制系統 | 最後數據備份 |
| 7 | 切斷所有電源 | 飛船靜止 |

### 2.2 預計關閉時間

| 階段 | 時間 | 備註 |
|:---|:---|:---|
| 準備工作 | 1 週 | 通知、計劃 |
| 反應爐熄火 | 1 天 | 逐步降功率 |
| 排空燃料 | 2 天 | 回收氘 |
| 排空冷卻劑 | 2 天 | 回收水 |
| 關閉控制系統 | 1 天 | 數據備份 |
| **總關閉時間** | **約 7 天** | |

## 5. 設備處置

### 5.1 處置選項

| 選項 | 說明 | 可行性 |
|:---|:---|:---|
| 1. 留原地 | 飛船靜止，不處理 | ✅ 最簡單 |
| 2. 可控墜落 | 導引至火星/月球表面 | ⚠️ 需額外燃料 |
| 3. 部分回收 | 回收有價值組件 | ❌ 成本極高 |
| 4. 推至墓地軌道 | 推至安全軌道 | ⚠️ 需額外燃料 |

### 5.2 推薦方案

**飛船留原地，關閉後靜止於火星軌道，作為「人類首艘核聚變飛船」紀念碑。**

| 優點 | 缺點 |
|:---|:---|
| 無需額外成本 | 留下太空垃圾 |
| 無技術風險 | - |
| 具歷史意義 | - |
| 可作未來參考 | - |
